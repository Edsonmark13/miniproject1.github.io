What is Javascript ?



Javascript is a programming language or a scripting language that allows the implementation of features on our pages.

Javascript is one of the 3 languages all web developers must learn: HTML, CSS and Javascript.

Javascript example is easy to code.

Comments are not executed buy the browser.

There two types of comments in Javascript, a single-line and the multi-line.

To make a single-line comment, add two forward slash // to the beginning to the line. To make all text following it to be a comment.

Another way is to use the /* */, the comment goes between the asterisks. This technique is much more flexible.

JS Variables



Javascript variable is simply a name of storage location. There are tow types of variables in Javascript : local variable an global variable.

The data stored could be a text, a number, boolean or others.


There are some rules while declaring a Javascript variable(also known as identifiers).

Name must start with a letter (a to z or A to Z), underscore(_), or dollar($)

After first letter we can use digits(0 to 9)

Javascript variables are case sensitive, for example i and I are different variables.

Reserved keywords (like var , function , if , etc. cannot be used as variable names.


Example

Correct Javascript Variables 

let name = "krisha";
let age = 10;

Incorrect Javascript Variables 
let 1Name = "krisha";
let %age = 10;


Tips: You should give meaningful names to your variable as much as possible, this helps maintain the readability of your code.

JS if-else



Javascript if else statement is used to execute the code whether condition is true or false.


There are three forms of if statement in Javascript

if it evaluates the content only if expression is true. 

if(condition){
    //execute code when the condition is true
}

Example 
let grade = 70;
if(grade >= 70){
    document.write('You passed!');
}

Output

You passed!


if else it evaluates the content whether the condition is true or false. 
if(condition){
    //execute code when the condition is true
}else{
    //execute code when the condition is false
}

Example 
let grade = 70;
if(grade >= 70){
    document.write('You passed!');
}else{
    document.write('You failed!');
}

Output

You passed!


if else if it evaluates the content only if the expression is true from several expressions. 
if(condition){
    //execute code when the condition is true
}else if(condition){
    //execute code when the condition is false
}else{
    //execute code when no condition is true
}

Example 
let grade = 70;
if(grade >= 90){
    document.write('Excellent!');
}else if(grade >= 80){
    document.write('Good Job!');
}else if(grade >= 70){
    document.write('You passed!');
}else{
    document.write('You failed!');
}

Output

You passed!

JS switch



Javascript switch statement matches an expression against the specified values in the case clauses.

Javascript switch statement is used to execute one code from multiple expressions. It just like else if statement that we have learned previous page. 

switch(expression){
    case value1:{
        //code to be executed
        break;
    }
    case value2:{
        //code to be executed
        break;
    }
    ...
    default:{
        //code to be executed when none are match from above values
        break;
    }
}

The switch statement evaluates the expression to value1 first, then to value2 and so on. If the expression matches a value, the associated code blocks are executed.

Example 
let grade = 'A';
switch(grade){
    case 'A':{
            document.write("Excellent!!");
            break;}
    case 'B':{
            document.write("Good Job!!");
            break;}
    case 'C':{
            document.write("Passed!!");
            break;}
    default:{
            document.write("Failed!!");}
}

Output

Excellent!!


The switch statement is fall-through i.e all the cases will be evaluated if you don't use break statement.

Example 
let grade = 'A';
switch(grade){
    case 'A':{
            document.writeln("Excellent!!");}
    case 'B':{
            document.writeln("Good Job!!");}
    case 'C':{
            document.writeln("Passed!!");}
    default:{
            document.writeln("Failed!!");}
}

Output

Excellent!!Good Job!!Passed!!Failed!!

JS Loops



Javascript loops are used to iterate the piece of code using the for , while , do while;

for loop iterates the elements for the fixed number of times. I should be used if the number of iteration is known. 

for(initialization; condition; increment/decrement){
    //code to be executed
}

Example 
for(i = 0; i < 5; i++){
    document.write(i + ' ');
}

Output

0 1 2 3 4


while loop iterates the element for the infinite number of times. It should be used if number of iteration is not known. 
while(condition){
    //code to be executed
}

Example 
let i = 0;
while(i < 5){
    document.write(i + ' ');
    i++;
}

Output

0 1 2 3 4


do while iterates the elements for the infinite number of times like while loop. But, code is executes at least once whether condition is true or false. 
do{
    //code to be executed
}while(condition);

Example 
let i = 0;
do{
document.write(i + " ");
i++;
}while(i < 5)

Output

0 1 2 3 4

JS Functions



Javascript functions are one of the fundamental building blocks of code.

It is a set of statements that performs a task or produces a value.

It is executed when it is invoked(called) by something.


Function Definition

Javascript function is defined or declared using the function keyword followed by:

The name of the function

A list of parameters (optional) enclosed in parentheses () . Multiple parameters are separated by comma , .

The block of codes or the statements enclosed in curly braces {} . 

function functionName(parameter){
    //code to be executed
}


Parameters act as placeholder variables inside the function.

When the function is called, the variables are assigned to the arguments (values) provided when the function is called.

Example 
function demo(str){
    document.write(str);
}

demo('Hello World');

Output

Hello World


Call a function

Function will only be executed when they are called or invoked. A function can be call when an event happens most commonly when a button is clicked.

Example 
function numberSquare(number){
    document.write(number * number);
}

Output




The return statement

The return statement is used in a function to stop its execution and to return a value to the function caller.

Example 
function numberSquare(number){
    return number * number;
}
document.write(numberSquare(4));

Output

16


Function Scope

Local variables are declared and can only be used inside the function.

If a variable is used outside the function, the value data type is undefined.

Example 
function numberSquare(){
    let number = 4;
    //variable number accessible
    document.write(number * number);
}
//the variable unaccessible
document.write(number);


Global variables can be used by any functions.

Global variables are variables declared outside a function.

Example 
let number = 4;
function numberSquare(){
    //variable number accessible
    document.write(number * number);
}
//variable number accessible

JS Strings



Javascript string is set of characters enclosed in quotes.

Javascript string can be enclosed in double quotes. Use single quotes if your string container double quotes.

Example 

let str1 = "Hello World";
let str2 = 'Hello "World"';

Javascript string can also be enclosed in single quotes. Use double quotes if your string contains single quote.

Example 
let str1 = 'Hello World';
let str2 = "Hello 'World";

Concatenating strings

To concatenate or to add string use the + operator;

Example 
let str1 = "Hello ";
let str2 = 'World"';
document.write(str1 + str2); //output Hello World

You can also use concatenate strings using the += operator;

Example 
let str = "Hello ";
str += 'World"';
document.write(str1); //output Hello World

Escaping characters

Escaping characters is crucial in handling strings.

It helps us make sure that our strings are recognized as pieces of text, not as part of the code.

To escape a characters, use a \ right before the character.

Example 
let str = "Hello \'World\'"; //escaping '


String Methods
String methods helps you when working with string.

Getting string length

To get the length or the number of characters in a string we use the length property.

Example 
let str = "Hello World";
document.write(str.length);//output 11


The length property is useful, for instance, you can use it to count how many characters a user has entered in one of their text fields.

Retrieving a character

We can retrieve a character from a string using the []. Simply put a [] after the variable name and write a number inside them.

Example 
let str = "Hello World";
document.write(str[4]);//output o


You may have expect that str[4] would return the character "l" but instead it returned the character "o" it is because Javascript count from 0 not from 1.

Finding string position

This can be done using the indexOf() method. It returns the index of the first occurrence of a substring in a string.

Example 
let str = "Please locate where 'locate' occurs!";
let strPos = str.indexOf("locate");
document.write(strPos); //output 7


While the indexOf() method. It returns the index of the last occurrence of a substring in a string.

Example 
let str = "Please locate where 'locate' occurs!";
let strPos = str.lastIndexOf("locate");
document.write(strPos); //output 21


Both indexOf() and lastIndexOf() returns "-1" if there is no occurrence found.

Searching a string

The search() method works similarly with the indexOf() method. The only difference is that indexOf() cannot take regular expressions as parameter.

The search() method is sensitively cased. To search a text insensitively, user the regular expression with the /i flag.

Example 
let str = "Hello World";
let strPos = str.search(/world/i); //case insensitive  output 0
let strPos1 = str.search("world"); //case sensitive   output -1


Don't forget the counting starts from 0. The search() method returns "-1" if the text is not found in the string.

Extracting a string

We can extract a substring from a string using the:

slice() method extracts a section of a string and returns it as a new string. It does not modified the original string. It takes two numerical parameters: start position andd the end position. It can take negative values.

Example 
let str = "Hello World";
let result = str.slice(0, 5);
document.write(result);

Output

Hello


substring() works the same as the slice() method. The only difference is that the substring() method cannot take negative values.

Example 
let str = "Hello World";
let result = str.substring(0, 5);
document.write(result);

Output

Hello


substr() method extracts a section of a string, from the specified index. It takes two numerical parameters: the starting index, anf the length of the string t be extracted.

Example 
let str = "Hello World";
let result = str.substr(6, 5);
document.write(result);

Output

World


replace() method replaces the specified text inside a string and returns a new string. It does not modified the original string. It takes two parameters : the string to replace and the string replacement.

To replace all the matching strings, use a regular expression with the /g.

Example 
let str = "Hello World";
let result = str.replace("o", "i"); //first occurrence
document.write(result + "<br>");
result = str.replace(/o/g, "i"); //all occurrence
document.write(result);

Output

Helli World
Helli Wirld


toUpperCase() and toLowerCase() method replaces the specified text inside a string and returns a new string. It does not modified the original string. It takes two parameters : the string to replace and the string replacement.

The toUpperCase() methods convert the string to upper case.

The toLowerCase() methods convert the string to lower case.

Example 
let str = "Hello World";
let result = str.toLowerCase();
document.write(result + "<br>");
result = str.toUpperCase();
document.write(result);

Output

Helli World
Helli Wirld

The toString() method returns a string representing the specified number

JS Arrays



Javascript array is an object that represents a collection of data.

Array methods

The forEach() method is another way to loop over an array, or access each element's data. It executes a callback function for each array element.

Example 

let persons = ['person1', 'person2', 'person3'];
persons.forEach(function(value, index, listOfPersons){
    document.write(value + "<br>");
});

Output

person1
person2
person3


The join() method creates and returns a new string by combining all the elements. The elements are separated by the specified separator character. If none is specified, it separates the elements with commas.

Example 
let persons = ['person1', 'person2', 'person3'];
let result = persons.join("-");
document.write(result);

Output

person1-person2-person3


The split() method is used to convert string to an array. It takes a separator as its argument.

Example 
let str = "person1 person2 person3";
let persons = str.split(" ");
document.write(persons.join(" - "));

Output

person1 - person2 - person3


The indexOf() method returns the index of the first occurrent of an item in the array.

The lastIndexOf() method returns the index of the last occurrent of an item in the array.

JS Dates



Javascript date object allows you to work with dates. You can display timer on the webpage by the help of Javascript.

Date constructor

The new Date() constructor returns a date object.

These are some of the basic forms of the date constructor:

new Date() creates a Date representing the current date. 

let date = new Date();

new Date(milliseconds) creates a Date object representing the date milliseconds since January 1, 1970. 
let date = new Date(86400000);

new Date(dateString) creates a Date object representing the date from a date string. 
let date = new Date("June 30, 2000");


Date set methods

The Date.setTime() sets the date to the time represented by a number of milliseconds since January 1, 1970.

The Date.setFullYear() sets the year for specified date.

The Date.setMonth() sets the month for specified date.

The Date.setDate() sets the day of the month for specified date.

The Date.setHours() sets the hours for specified date.

The Date.setMinutes() sets the minutes for specified date.

The Date.setSeconds() sets the seconds for specified date.

The Date.setMilliseconds() sets the milliseconds for specified date.

set to assign and get to retrieve

Example
