What is CSS ?



CSS stands for Cascading Stylesheet

CSS describes the presentation of the web pages.

CSS describes how elements rendered on the screen.

Why Use CSS ?


CSS defines style of your web pages including design, layout and variation in display across devices.

CSS can save you a lot of work. An external stylesheet saved as .css can be used to style multiple web pages all at once.

Example 

<!DOCTYPE html>
<style>
    body{
        background-color: white;
    }
    .sample-button{
        background-color: lightblue;
        border: 1px solid black
    }
</style>
<button class="sample-button">sample</button>

Output


<!DOCTYPE html> tag defines the document type that the browser is going to render. This is used for legacy reasons. If you want to get to the latest version of HTML (HTML5) then it's recommended to use this tag.

<html> this is element encloses everything inside of an html document; it includes tags, elements, style sheets, scripts, text, multimedia and a lot more.

<head> this element encloses the metadata of a document which will not be displayed on the main content of a web page; this includes <style>, <script>, <title>, <meta> tags and a lot more.

<title> defines the title of a webpage.

<body> defines the document's body and encloses elements like <h1>, <p>, <img> and a lot more.

<h1> defines a heading

<p> element defines a "paragraph", a block of text that has a small amount of spacing in between its top and bottom.

HTML Tags


HTML Tags are element name surrounded by angle brackets

<p>This is a paragraph.</p>

<p> is an example of start tag or opening tag
</p> is an example of end tag or ending tag

HTML comments both help beginners and experienced developers to easily organized their codes

HTML Style



HTML Styles are used to style /design HTMl element it means changing the default values.

Add styles to an element, such as color, font, size, and more.

Internal Style


Internal Style is composed of one or more CSS rule set.

A CSS rule set consists of a selector, a property and a value surrounded by curly braces.

Syntax

<style>
    selector {
         property : value
    }
 </style>

Internal Style is included inside <head> tag.


Inline Style


Inline style is used to style elements using the stye attribute.

Syntax

<p style="property:value"></p>

Example backgroundand color property

<p style="background-color:green; color:white"> I am green</p>
<p style="background-color:red; color:white"> I am red</p>

Output

I am green

I am red

HTML Colors



HTML Colors define colors that we see on a web.

In this topic we will use inline style with CSS property color to change the text color of a HTML elements.

Color Name - defines any valid color by its color name.
Hex Color - defines any valid color by a hash followed by letters and/or numbers.
RGB Color - defines any valid color by this format rgb(number, number, number)
RGB stands for Red, Green and Blue.

Example


<p style=" color : green; "> I use Color Name to change my text color</p>
<p style=" color : #4DCF08"> I use Hex Color value to change my text color</p>
<p style=" color : rgb(77, 207, 8)"> I use RGB Color value to change my text color</p>

Output

I use Color Name to change my text color

I use Hex Color value to change my text color

I use RGB Color value to change my text color

HTML Links



HTML Links are very necessary for all web pages.

All website have links. HTML Links are also called hyperlinks.

HTML Links help users jump from their current web page location to another.

To make an actual link using the <a> tag, use the href attribute.

It look link this : <a href="index.html">Link</a>

Internal / Relative Links



Linking inside same directory.

It look link this : <a href="file_in_the_same_folder.html">Link</a>


External / Absolute Links


Linking outside directory

It look link this : <a href="../editor/index.html">B’ Academy editor</a>

or : <a href="https://www.google.com">Go to google</a>


Link target attribute

The target attribute specifies where to open the linked document.

_self Default. Opens the document in the same window/tab as it was clicked.

_blank Opens the document in a new window or tab.

_parent Opens the document in the parent frame.

_top Opens the document in the full body of the window.


Example


<a href="../editor/index.html" >B' Academy editor</a><br>
<a href="https://www.google.com" target="_blank" >Go to google</a>

Output

B' Academy editor

Go to google

HTML Audio



To embed audios in HTML we have to use the <audio> tag with its attribute src or the <source> tag with its src attribute.

The controls attribute adds audio controls, like play, pause, and volume.

The autoplay attribute start an audio file automatically.

The muted attribute start audio file but muted.

The loop attribute will start audio over and over again in an endless loop.

There are three supported audio formats: MP3, WAV, and OGG

The muted, loop, controls and autoplay are boolean they don't need values.

Example


<audio src="./audio/sample3.ogg" controls ></audio>

Output

HTML Division



The <div> tag is known as Division tag.

The <div> tag used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc).

Div tag is Block level tag.

It is used to the group of various tags of HTML so that sections can be created and style can be applied to them.

It is a generic container tag.

The <div> tag is easily styled by using the class or id attribute.

Any sort of content can be put inside the <div> tag!


Example

<div style="height: 30px; text-align: start; color: white; background-color: black">div #1</div>
<div style="height: 50px; text-align: center; color: white; background-color: green">div #2</div>
<div style="height: 30px; text-align: end; color: white; background-color: lightblue">div #3</div>

Output

div #1
div #2
div #3

HTML Form



HTML Forms can be used to collect user data.

The <form> tag defines a HTML Form.


HTMLForm Attributes


autocomplete tells the browser to automatically complete the form.

name, id and class used for identifying the form; for the purpose of styling.

target indicates the form where to display the response. It can be _self, _blank , _parent or _top

action where the input data processed.

method the HTTP method to submit the form with. The value can be POST method the data sent as the request body or

GET the data appended to the action URL.


Example

<form action="index.html" method="GET">
<label for="name">Full name</label>
<input type="text" name="name"  id="name"><br>
<label for="email">Email Add</label>
<input type="text" name="email"  id="email"><br>
<label for="message">Messages</label>
<input type="text" name="message"  id="message"><br>
<input type="submit" value="submit">
</form>

Output





To submit a form a user has to click the submit button. Take a look at the URL as you can see the data you enter is now appended to its original url.

GET method must not be used when submitting personal data.
